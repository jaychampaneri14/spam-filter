# Spam Filter

> NLP spam email/SMS classifier using TF-IDF features with Naive Bayes and Logistic Regression

## Overview
This project is part of a comprehensive AI/ML portfolio covering natural language processing and text classification.

## Features
- Production-ready implementation
- Well-documented code with examples
- Easy to customize and extend

## Installation

```bash
git clone https://github.com/jaychampaneri14/spam-filter.git
cd spam-filter
pip install -r requirements.txt
python main.py
```

## Requirements
See `requirements.txt` for full dependencies.
Main libraries: scikit-learn, numpy, pandas

## Usage
```bash
python main.py
```
Run with `--help` for full options.
